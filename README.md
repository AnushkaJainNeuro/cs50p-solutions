# cs50p-solutions
My solutions to Harvard's CS50P (Introduction to Programming with Python) 
problem sets — summer 2026.

## why I'm doing this
CS50P is the foundation layer before I get to the work I actually care about: 
EEG signal processing and neural decoding for BCI systems. I'm documenting 
this publicly because the habit of shipping code daily matters as much as 
the code itself.

## structure
Each folder corresponds to a CS50P unit. Where possible I've included my 
first attempt alongside my rewrite — the rewrite is always done from memory 
without looking at the original.
